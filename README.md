# AetherOS

**A cinematic, monochrome operating system built from flowing waves, darkness, light, and elegant motion.**

This repository contains the design vision, initial website skeleton, and assets for AetherOS — a minimal, atmospheric desktop experience that emphasizes motion, mood, and a premium, mysterious aesthetic.

## Vision

AetherOS is designed to feel less like a conventional computer and more like entering another world. Every interaction should feel cinematic, mysterious, and futuristic.

Key goals:
- Atmosphere first: every screen and transition contributes to mood.
- Minimalism and focus: only essential UI elements are shown.
- Motion as identity: flowing waves, particles, and subtle energy define the system.
- Performance: visual beauty must not compromise responsiveness.

## What’s included

- docs/index.html — A small landing page prototype with animated monochrome waves.
- docs/styles.css — Styling for the landing page (dark, frosted panels, centered logo).
- docs/scripts/waves.js — Lightweight canvas-based wave animation used on the landing page.

## Design specification

This repository contains the full design brief inside `DESIGN.md`. The design emphasizes:
- Pure Black, Charcoal Gray, Deep Gray, Silver, White.
- Thin-to-medium typography with generous spacing.
- Smooth, slow, cinematic animations (60–120 FPS target when possible).
- Frosted glass windows, rounded corners, and soft white glows.

See DESIGN.md for the complete specification and the boot, installer, and desktop sequences.

## Development

- The docs/ folder contains a minimal static prototype you can open locally (open docs/index.html in your browser).
- The canvas animation is intentionally lightweight and configurable.
